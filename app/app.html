This importer will scan for only your gear. Instructions:

  1. Install requirements from the instructions on Github

  2. Leave your emulator open, and close Epic 7

  3. Click the Start scanning button below

  4. Open Epic 7, and load into the lobby

  5. Optionally, if you want to load your storage items

  8. Import the 'gear.txt' with the Merge option

  1. Install requirements from the instructions on Github

  2. Leave your emulator open, and close Epic 7

  3. Click the Start scanning button below

  4. Open Epic 7, and load into the lobby

  5. Optionally, if you want to load your storage items

  6. Click the Stop scanning button below

  7. Wait up to 30 secs, then once the data appears, click Export to save it to a 'gear.txt'

  8. Import the 'gear.txt' with the Merge option

  9. If the message tells you to, use the Level = 0 filter to manually fix any level 0 items on the Gear tab
